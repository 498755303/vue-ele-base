### 图钉组件

- Attributes

| 参数          | 类型   | 是否必须 | 说明                             | 默认值 |
| ------------- | ------ | -------- | -------------------------------- | ------ |
| offset-top    | Number | 否       | 距离窗口顶部达到指定偏移量后触发 | 0      |
| offset-bottom | Number | 否       | 距离窗口底部达到指定偏移量后触发 | -      |

- Events

| 参数      | 说明                     | 回调参数 |
| --------- | ------------------------ | -------- |
| on-change | 在固定状态发生改变时触发 | true     | false |
